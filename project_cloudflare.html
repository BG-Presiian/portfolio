CloudFlare Security Scanner

Technologies

I used a github repository to make changes more easily and have a manageable directory. I used a virtual machine with a Kali Linux disc. Kali linux terminal was used to run code (sudo ./netdump > net-dump.txt). The language used was C, as this was low level programming and would've been more difficult in a middle or upper level language. I used vim to write my code.

Description

I would decode a packet I received, a packet may look something like:
"6c40 0889 c448 f832 e4a7 bf38 0800 45000034 4749 4000 3806 9360 a27d 0303 c0a801f2 01bb d2f1 6c7b 2f45 d5e0 061b 80110027 0b85 0000 0101 080a 4a1f 61e4 4a46c042"
I decoded these hexadecimal values and formatted it so that it would display corresponding values of a TCP/IP header on a 32 bit system.
